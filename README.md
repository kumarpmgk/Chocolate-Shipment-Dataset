# Chocolate-Shipment-Dataset
Developed an interactive Power BI report using a real-world shipment dataset.
Performed data cleaning and transformation using Power Query, built a star schema data model, and established relationships between multiple tables.
Applied DAX concepts and custom measures to support advanced calculations such as Year-over-Year comparisons and KPI tracking. Designed and implemented a calendar table to enable time-intelligence analysis.
Created business-focused dashboards using a variety of visuals including line charts, donut and pie charts, treemaps, KPI cards, and formatted tables. Implemented slicers, visual/report filters, Top N analysis, conditional formatting, and customized tooltips to enhance usability and interactivity. Incorporated images via URLs for user experience and decision-making support.
